# Security Policy

MiMo Airdrop Scout is a defensive educational tool. It does not require private keys, seed phrases, wallet signatures, KYC documents, or production API keys to run.

## Reporting issues

If you find a bug or unsafe recommendation, open a GitHub issue with:

- command/input used
- expected output
- actual output
- why the recommendation may be risky

Do not include private keys, seed phrases, wallet addresses tied to sensitive activity, or API tokens in issues.
